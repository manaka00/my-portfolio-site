概要

藤女子中学校・高等学校の公式Webサイトです。
学校の概要、学生生活、進路、入学案内など、藤女子中学校・高等学校に関する情報を掲載しています。

制作期間（担当期間）

2022年1月～2022年8月

担当範囲

CMS移行 / 制作 / 運用 / 運営

業務内容・工夫した点

CMSの移行作業、既存ページの修正や新規ページの作成を担当していました。
また、公開後も継続的にサイトの運用・運営を行っていました。

学生や保護者、学校関係者など幅広い方々が利用するサイトのため、見た目の調整やページ遷移時のエラー防止など、丁寧な制作を心がけました。
学校の生徒募集に関わる重要なサイトであるため、修正依頼には迅速に対応できるよう意識しました。

使用言語・ツール

言語：HTML / CSS / JavaScript / Ruby
ツール：学校HP楽々プログラム（学校CMS）※1 / illustrator / Photoshop
