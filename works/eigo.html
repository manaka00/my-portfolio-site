英語教室情報

HP

概要

自社で運営している英会話スクールを紹介しているサイトです。
学ぶ目的やレッスン形式などの「こだわり条件」から英会話スクールを探すことができます。
こども向けから大人向けまで幅広いスクールを掲載しています。

制作期間（担当期間）

2021年9月～現在に至る

担当範囲

開発 / 運用 / 運営

業務内容

既存ページやシステムの修正・運用・運営、SEOタグの設定を行っています。

学生や保護者、英会話スクール関係者など幅広い方々に利用されているサイトであるため、見た目の調整や検索・ページ遷移時にエラーが発生しないよう、丁寧な開発を心がけています。

使用言語・ツール

言語 : HTML / CSS / JavaScript / Ruby / Java
DB : MySQL
ツール : CMS(会社独自) / NetBeans / XD
