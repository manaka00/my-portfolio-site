Our products

学生投稿動画

アプリ

概要

自社が運営する学校情報ポータルサイト「日本の学校」の学校紹介ページに掲載する動画を撮影・編集できるアプリです。
学校に通う学生が自ら撮影し、編集した動画を投稿することで、在校生のリアルな声や学校の魅力を発信できます。

※本サービスは終了しました

制作期間（担当期間）

2022年9月～2023年2月

担当範囲

開発

業務内容

Android版スマートフォンアプリケーションの開発を担当しました。
また、投稿動画を管理するための管理画面の開発も行いました。

会社として初の大規模アプリ開発であったため、チーム内での情報共有を重視し、円滑な連携によってプロジェクトを完遂しました。

使用言語・ツール

言語 : kotlin / HTML / CSS / JavaScript / Ruby / Java
DB : MySQL
ツール : CMS(会社独自) / Android Studio / XD

機能概要

図1　トップ画面（学校情報登録前）

図2　動画編集画面

図3　トップ画面（学校情報登録後）

図1は、本アプリのトップ画面です。
右下のカメラアイコンをタップすると、動画の撮影が開始します。

図2は、撮影した動画を編集する画面です。
トリミングやテキストを挿入することができます。

図3は、所属する学校を登録した後のトップ画面です。
過去に作成した動画や同じ大学に通う人が作成した動画が表示されます。
アプリで作成した動画は、自社が運営するWebサイト「日本の学校」に掲載されます。
