日本の学校

HP

概要

自社で運営している学校情報のポータルサイトです。
高校生を対象とした大学・短期大学・専門学校の進学情報を中心に、高校、中学校の情報まで網羅しています。

制作期間（担当期間）

2021年4月～現在に至る

担当範囲

開発 / 運用 / 運営

業務内容

既存ページやシステムの修正・運用・運営、新規ページやシステムの開発、SEOタグの設定を行っています。

学生や保護者、学校関係者など幅広い方々に利用されているサイトであるため、見た目の調整や検索・ページ遷移時にエラーが発生しないよう、丁寧な開発を心がけています。
また、システム部だけでも約10名が常に開発や運用保守に携わっているため、ソースコードや引継ぎ文書は誰が見ても分かりやすいように記述することを意識しています。

使用言語・ツール

言語： HTML / CSS / JavaScript / Ruby / Java / Go
DB：MySQL
ツール：CMS(会社独自) / Visual Studio Code / NetBeans / XD
