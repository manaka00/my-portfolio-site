OC音声案内アプリ

アプリ

概要

オープンキャンパスなどのイベントで使用する音声案内アプリです。
事前に登録した施設や設備の情報を、AI音声が自動で読み上げて案内してくれます。
来場者がスマートフォンを通して、校内をスムーズに見学できるように設計されています。

※アプリケーションストアでインストールできますが、サービスは終了しています

制作期間（担当期間）

2023年1月～2023年5月

担当範囲

開発（Android版）

業務内容

Android版スマートフォンアプリケーションの開発を担当しました。
また、原稿を管理するための管理画面の開発も行いました。

会社として2番目に大規模なアプリ開発であったため、チーム内での情報共有を重視し、円滑な連携によってプロジェクトを完遂しました。

使用言語・ツール

言語 : kotlin / HTML / CSS / JavaScript / Ruby / Java
DB : MySQL
ツール : CMS(会社独自) / Android Studio / XD

URL

Google Play

機能概要

図1　施設情報読み取り画面

図2　施設一覧画面

図3　音声案内画面

図1は、施設情報読み取り画面です。
学校から配布された専用のQRを読み取ります。

図2は、QRの読み取りが成功した後に表示される、学校の施設や設備一覧画面です。
詳しく知りたい施設や設備を選択します。

図3は、施設や設備の音声案内画面です。
「音声案内を再生する」ボタンをタップすると、AI音声で施設情報を紹介してくれます。
